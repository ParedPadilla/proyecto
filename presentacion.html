Web of Things

Thing Description

 WoT Thing Description (TD) es una parte o fragmento importante de lo que se conoce como el Web Of Things.
El Thing Description se puede considerar como el punto de entrada de la Cosa (Thing) con la cual se está conectando o interactuando, así como un index.html en una pagina web.
El TD tiene cuatro componentes:

  1. Metadata textual sobre la Cosa propia
    Dos de los campos de esta metadata son escritos de manera que el humano pueda leerlos y se pueden usar para enviarlos a un display.
  2. "Interaction Affordances"
    Un conjunto de capacidades de la Cosa que indican como se puede usar la Cosa.
    • Propiedades: Exponen el estado de la Cosa
    • Acciones: Invoca una funcion de la cosa o comienza proceso en la Cosa.
      (ej. apagar/prender lampara o bajar la intensidad de la luz a cierta hora)
    • Eventos: Describe un evento y con esto se puede enviar al consumidor algún tipo de alerta.
      (ej. lampara sobrecalentandose)
  3. "Schemas" para los datos intercambiados al conectarse.
  4. Enlaces de paginas web, para describir relaciones con otras Cosas o documentos en la web.
